# Practitioner is the listing unit

The marketplace lists individual healthcare practitioners, not practices. Verification anchors on the regulator-registered person (GOC for Phase 1), reviews and trust signals attach to humans rather than buildings, and the £29/mo subscriber is the Practitioner. **Practice** is an optional zero-or-one attribute on a Practitioner; if Phase 2 demands many Practitioners per Practice (e.g. Dental Hygienist working in someone else's surgery), it will require a schema change rather than a model rework.
